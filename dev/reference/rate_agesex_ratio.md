# Rate the acceptability of the age and sex ratio test p-values

Rate the acceptability of the age and sex ratio test p-values

## Usage

``` r
rate_agesex_ratio(p)
```

## Arguments

- p:

  A vector of class `double` of the age or sex ratio test p-values.

## Value

A `character` vector with the same length as `p` for the acceptability
rate.
