# mwana: An Efficient Workflow for Plausibility Checks and Prevalence Analysis of Wasting in R

A simple and streamlined workflow for plausibility checks and prevalence
analysis of wasting based on the Standardized Monitoring and Assessment
of Relief and Transition (SMART) Methodology
<https://smartmethodology.org/>, with application in R.
