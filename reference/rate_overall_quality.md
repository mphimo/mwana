# Rate the overall acceptability of the data

Rate the overall data acceptability score into "Excellent", "Good",
"Acceptable" or "Problematic".

## Usage

``` r
rate_overall_quality(q)
```

## Arguments

- q:

  A `numeric` or `integer` vector of data acceptability scores.

## Value

A vector of class `factor` with the same length as `q` of overall rate
of acceptability of the data.
