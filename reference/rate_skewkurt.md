# Rate the acceptability of the skewness and kurtosis test results

Rate the acceptability of the skewness and kurtosis test results

## Usage

``` r
rate_skewkurt(sk)
```

## Arguments

- sk:

  A vector of class `double` for skewness or kurtosis test results.

## Value

A vector of class `factor` with the same length as `sk` for the
acceptability rate.
